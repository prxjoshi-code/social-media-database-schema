# Social Media Database Schema

## Overview
Designed and implemented a relational SQL database schema modeling a simplified social media platform, including users, posts, comments, likes, and friendships.

## What I Built
- Normalized database schema with core social media entities and relationships
- SQL scripts for table creation, constraints, and sample data population
- Stored procedure to update post like counts
- User-defined function to compute average likes per user
- Cursor-based SQL example for iterative data processing

## Tech Stack
- SQL (Microsoft SQL Server)
- Relational Database Design
- Stored Procedures, Functions, Cursors

## Context
This project was completed as part of a graduate-level database systems course to demonstrate data modeling, relational integrity, and procedural SQL concepts.
